Datasets and Download

Please find here all links to both the raw and processed data sets.
Examining processed data are found in the Tutorial section.

Raw data

  • MSI data in .imzml format can be found on METASPACE

  • HybISS data (spot detections as x,y coordinates) per gene can be found on Zenodo

  • All bulk LC/MS data are available as Excel sheets (Supplementary Table 2) on Zenodo

Processed data

  • 4D lipidomic data available as .h5ad files. Anatomical annotations available for 72 hpf zebrafish (see tutorial page). Metadata include m/z, lipid species, lipid class from Zenodo

  • HybISS data as spots in the coordinate space of its consecutive MALDI-MSI section (with corresponding lipid data from the MSI experiment). Available on Zenodo

  • Supplementary Table that includes all detected lipids with metadata (Supplementary Table 1). Available on Zenodo
